# Reservas (CoverManager)

La web usa el mismo motor que WordPress: **CoverManager**, restaurante `restaurante-wearecacho`.

## Página

- `reservas.html` — iframe embebido
- Config: `reservas-config.js`
- Idioma del motor: `reservas.js` (ES → `spanish`, CA → `catalan`, EN → `english`)

## Gestión de mesas y horarios de reserva

No sale de Google Sheets. Se configura en el panel de [covermanager.com](https://www.covermanager.com) (misma cuenta que el plugin de WordPress).

## Enlaces

Todos los botones «Reservar» del sitio apuntan a `./reservas.html`.
